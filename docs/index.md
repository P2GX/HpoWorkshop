# :material-dna: HPO Workshop

!!! abstract ""
    Welcome to the **Human Phenotype Ontology (HPO) Workshop**.

    This site provides hands-on material to help you learn how to use the  
    [Human Phenotype Ontology (HPO)](https://hpo.jax.org/app/){:target="_blank"}  
    in bioinformatics and clinical workflows.

    The content is designed for guided workshops but can also be used
    independently for self-study.

---

## :material-school: What you will learn

<div class="grid cards" markdown>

- :material-format-list-checks: **Phenotype Annotation**  
  Learn how to describe patient phenotypes using structured HPO terms.

- :material-file-document-outline: **Phenopackets**  
  Create and explore interoperable phenotype datasets.

- :material-database-search: **Exomiser Analysis**  
  Run phenotype-driven variant prioritisation workflows.

- :material-lightbulb-on-outline: **Best Practices**  
  Apply HPO effectively in research and clinical environments.

</div>

!!! note "Preparing for the workshop"

    This workshop combines short lectures with practical exercises.
    It is intended for **bioinformaticians, clinicians, and translational researchers**
    working with the HPO or related software.

    If you would like to prepare in advance:

    - Read the [background articles](./introduction/reading.md){:target="_blank"}

    There will be many activities in the workshop. Optionally, users can run Exomiser on their own laptops. If you would like to do this, please

    - Install a [Java runtime environment](./exomiser/java.md)
    - Download the Exomiser [software and data files](./exomiser/cmdline.md)

      The data files are roughly **20 GB**, so please download them before
      the workshop if you plan to run Exomiser locally.
