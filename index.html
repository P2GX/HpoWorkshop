    HPO Workshop

    Welcome to the Human Phenotype Ontology (HPO) Workshop.

    This site provides hands-on material to help you learn how to use the
    Human Phenotype Ontology (HPO)
    in bioinformatics and clinical workflows.

    The content is designed for guided workshops but can also be used independently for self-study.

    What you will learn

    • Phenotype Annotation
      Learn how to describe patient phenotypes using structured HPO terms.

    • Phenopackets
      Create and explore interoperable phenotype datasets.

    • Exomiser Analysis
      Run phenotype-driven variant prioritisation workflows.

    • Best Practices
      Apply HPO effectively in research and clinical environments.

    Preparing for the workshop

    This workshop combines short lectures with practical exercises. It is intended for bioinformaticians, clinicians, and translational researchers working with the HPO or related software.

    If you would like to prepare in advance:

    • Read the background articles
    • Download the Exomiser software and data files

    The data files are roughly 20 GB, so please download them before the workshop if you plan to run Exomiser locally.
